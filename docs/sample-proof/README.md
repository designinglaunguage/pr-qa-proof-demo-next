# Sample QA Proof

고객에게 GitHub Actions 실행 전에도 바로 보여줄 수 있는 샘플 proof 복사본입니다.

- `sample-pr-comment.md`: PR 댓글에 올라갈 요약 본문
- `qa-proof-report.md`: PM/CTO용 요약과 체크 결과 리포트
- `screenshots/`: 홈, 가격/비교, 설정 화면의 desktop/mobile screenshot

실제 workflow 실행 결과는 `qa-proof-artifacts/`에 생성되고, 이 폴더는 데모 repo 안에 보존되는 샘플입니다.
